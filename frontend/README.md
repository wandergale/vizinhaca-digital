# Vizinhança Digital — Frontend

React + Vite application for the Vizinhança Digital platform.

## Getting started

```bash
npm install
npm run dev
```

The app will be available at `http://localhost:5173`.


## Routes

| Path | Page |
|---|---|
| `/autenticacao` | Authentication screen |
| `/cadastro-acoes` | Action registration |
| `/calendario` | Event calendar |
| `/minhas-inscricoes` | User registrations |
| `/inscricao-voluntario` | Volunteer sign-up |
| `/painel-inscricoes` | Registrations panel |
| `/relatorios` | Reports |
